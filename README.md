# nasbeery
Installer for NasBeery ZFS NAS for Raspberry PI 4 and 400 on Raspberry Pi OS Lite 64-Bit

```bash
wget -O setup-nasbeery https://github.com/bashclub/nasbeery/raw/main/setup-nasbeery
bash setup-nasbeery
```
